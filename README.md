# MapTool

## 조작키
W, A, S, D: 이동  
Q: 상승  
E: 하강  
마우스 우클릭후 드래그: 회전  
←, →, ↑, ↓: 회전  
F1; UI 끄기 켜기  
F2: Wireframe 모드 끄기 켜기  
F3: 상호작용 재생 끄기 켜기  
F4: CellLine 표시 끄기 켜기  
F5: 자유이동 끄기 켜기
